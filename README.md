# Actividad 1 - Programación Web

En esta actividad se subieron varios ejercicios de HTML, CSS y JavaScript realizados en clase.

La actividad consiste en juntar todos los ejercicios en un solo repositorio de GitHub y crear un archivo `index.html` que funcione como menú principal para poder entrar a cada ejercicio.

También se publica el repositorio con GitHub Pages para que los ejercicios se puedan ver desde internet y no solamente desde localhost.

## Contenido

- Ejercicios básicos de HTML
- Uso de enlaces e imágenes
- Uso de tablas
- Formularios
- Estilos con CSS
- Funciones sencillas con JavaScript
